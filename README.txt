    Build instructions
------------------------
1) install node
2) sudo npm install -g webpack
3) clone repo
4) cd into it; npm install
5) webpack

Note: If "webpack" does not work, try "npm run build"



      Controls
------------------------
       w - Jump
       s - Crouch
       a - Walk left
       d - Walk right
       q - Dash
spacebar - Spit hairball
